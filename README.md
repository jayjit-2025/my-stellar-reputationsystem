# 🧩 Reputation System on Stellar (Soroban)

## 📌 Project Description

This project is a decentralized reputation system built using Soroban smart contracts on the Stellar network. It allows users to maintain and manage reputation scores in a transparent and immutable way.

The goal is to create a foundational layer for trust-based applications such as marketplaces, communities, and decentralized platforms.

---

## ⚙️ What it does

This smart contract enables:

* Creating a reputation profile for users
* Increasing reputation based on positive actions
* Decreasing reputation based on negative actions
* Fetching a user's current reputation score

All data is stored on-chain using Soroban's persistent storage.

---

## 🚀 Features

* 🧑 User-based reputation tracking
* ➕ Increment reputation score
* ➖ Decrement reputation score
* 🔍 Query reputation score
* 💾 Persistent on-chain storage
* ⚡ Lightweight and efficient contract logic
